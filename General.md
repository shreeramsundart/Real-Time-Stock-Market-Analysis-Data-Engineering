### SnowFlake:

## What is Snowflake?
**Snowflake** is a **cloud-based data warehousing platform** used to **store and analyze large amounts of data**.  
It runs as a **SaaS service** on AWS, Azure, or Google Cloud.

---

## Why Snowflake?
- Fully cloud-based (no server management)
- Separates **storage and compute**
- High performance & auto-scaling
- Easy to use (SQL-based)
- Strong security & compliance
- Supports structured and semi-structured data
- Pay-as-you-go pricing

---

## Key Advantage
**Storage and compute are independent**, allowing multiple users to analyze the same data simultaneously with better performance and lower cost.

### dbt (Data Build Tool)

**dbt** is a **data transformation tool** used in analytics engineering.  
It helps transform raw data in your **data warehouse** into **clean, modeled, and analyzable tables** using **SQL**.

### Key Features
- SQL-based transformations
- Version control with Git
- Automated documentation
- Testing and data quality checks
- Works with warehouses like Snowflake, BigQuery, Redshift

### Why Use dbt?
- Keeps data clean and reliable
- Makes analytics reproducible
- Easy collaboration for data teams
